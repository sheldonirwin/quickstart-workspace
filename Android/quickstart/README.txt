Quickstart is a simple sample application written by Sheldon Irwin <www.sheldonirwin.me>

It was created as a simple quickstart app, showing the simplest pom structure to get started.

To build Quickstart:
	mvn clean install

To run Quickstart:
	mvn clean install android:deploy android:run

To run Quickstart on an emulator, start the emulator as follows:
	mvn android:emulator-start
